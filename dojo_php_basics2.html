Basics 2

Commence par regarder les 9 films Star Wars (et les spin-off en option), si ce n'est pas déjà fait

Démarrage

  1. Reprends le fichier index.php
  2. Démarre un serveur HTTP sur le dossier basics

Boucle for

  1. Affiche de 0 à 9 inclus via une boucle
  2. Affiche de 0 à 10 inclus via une boucle
  3. Affiche de 1 à 10 inclus via une boucle
  4. Affiche de 10 à 0 inclus via une boucle
  5. Affiche de 25 à 50 inclus, tous les 5
  6. Affiche les valeurs possibles entre 10 à -10, tous les 3

Conditions

Soit $name = 'Darth Vader';

  1. Ecris une condition pour tester si $name égal à "Darth Vader" ou "Darth Sidious", affiche "Sith" dans ce cas
  2. Sinon si $name égal "Yoda", affiche "Jedi"
  3. Sinon affiche "sans doute une personne sans pouvoir"

Tableau indexé

Soit les valeurs : 'Obi-Wan Kenobi', 'Yoda', 'Luke Skywalker', 'Windu', 'Qui-gon Jinn'

  1. Construis un tableau $jedis (indexé numériquement)
  2. Ajoute 'Rey' dans le tableau
  3. Affiche uniquement le plus vieux jedi
  4. Boucle pour afficher tous les jedis

Tableau associatif

Créer un tableau associatif avec le nom en clé et l'age en valeur :

  • Yoda 900
  • Leia 19
  • Luke Skywalker 19
  • Darth Vader 46
  • Chewbacca 200
  1. Ajoute Han Solo, 29ans, au tableau
  2. Classe le tableau du plus jeune au plus vieux, puis affiche les noms et ages dans une liste HTML
  3. Affiche les noms et ages des personnages par ordre alphabétique dans une liste HTML
  4. Affiche uniquement les personnages ayant plus de 100ans (affiche leur age)

Tableaux multidimensionnels

Soit les personnages :

  • Jedi : Luke Skywalker, Yoda, Windu
  • Sith : Darth Vader, Darth Maul, Dooku
  • Gungan : Jar Jar Binks
  • Human : Han Solo, Leia

Créé un tableau associatif avec le nom des races en clé, et un tableau de personnages de cette race en valeur

  1. Boucle sur le tableau afin d'afficher les noms de race dans une balise h2, suivi d'une liste pour les noms de cette race
  2. Boucle sur le tableau pour afficher dans une liste, la race et le nombre de personnages de cette race
