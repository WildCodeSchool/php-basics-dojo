Conditions

Soit $name = 'bilbo';

  1. test si $name égal "bilbo" ou "gollum", affiche "hobbit"
  2. sinon si $name égal "gandalf", affiche "magicien"
  3. sinon affiche "sans doute un humain"

Tableau associatif

Créer un tableau en tableau associatif avec le nom en clé et l'age en valeur :

  • Aragorn 90
  • Gimli 100
  • Gandalf 850
  • Legolas 650
  • Frodo 35
  1. Ajouter Boromir, 40ans au tableau
  2. Classer le tableau du plus jeune au plus vieux, puis afficher les noms et ages dans une liste HTML
  3. Afficher les noms et ages des personnages par ordre alphabétique dans une liste HTML
  4. Afficher uniquement les personnages ayant plus de 100ans (afficher leur age)

Soit les personnages :

  • Hobbit : Frodo, Bilbo, Merry, Pippin
  • Elve: Legolas, Galadriel, Elrond
  • Dwarf : Gimli, Gloïn, Thorin
  • Human : Aragorn, Boromir

Créé un tableau associatif avec le nom des races en clé, et en valeur, un tableau de personnages de cette race

  1. Boucler sur le tableau afin d'afficher les noms de race dans une balise h2, suivi d'une liste pour les noms de cette race
  2. Boucler sur le tableau pour afficher dans un liste, la race et le nombre de personnages de cette race
