FONCTION

  1. Crée une fonction nommée sayHello permettant d'écrire "Hello Nite Owl".

RETOUR

Une fonction ne doit jamais (sauf cas rare) afficher directement quelque chose (echo, print_r, var_dump...).
Une fonction doit retourner une valeur d'un type bien défini.

En ré-utilisant notre fonction "sayHello()"

  1. Modifie la fonction pour qu'elle retourne "Hello Nite Owl" plutôt qu'elle l'affiche directement.

PARAMETRE

Toujours à partir de notre fonction "sayHello()"

  1. Modifie la en lui ajoutant un paramètre nommé "name" de type string
  2. Modifie le retour de la fonction pour qu'elle retourne "Hello {name}"
  3. Rend le paramètre "name" optionnel. Si rien n'est précisé, ta fonction devra retourner "Hello Dr. Manhattan"
  4. Algo : Retourne "Hello {name}" uniquement si {name} est un watchmen :
    ('Dr. Manhattan', 'Ozymandias', 'Silk Spectre', 'Rorschach', 'The comedian', 'Nite Owl')
